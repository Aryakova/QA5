# Виды дефектов программного обеспечения

| Вид дефекта                | Краткое описание |
|----------------------------|------------------|
| Функциональный дефект      | Некорректная или отсутствующая реализация заявленной функциональности. |
| Дефект интерфейса (UI)     | Ошибки в отображении элементов интерфейса, несоответствие макетам, неправильные шрифты, цвета, размеры. |
| Дефект юзабилити           | Неудобство использования, нелогичное поведение, сложность для пользователя. |
| Дефект производительности  | Медленная работа приложения, долгие отклики, утечки памяти, чрезмерное потребление ресурсов. |
| Дефект безопасности        | Уязвимости, позволяющие несанкционированный доступ, SQL-инъекции, XSS, CSRF и др. |
| Дефект совместимости       | Некорректная работа на определённых устройствах, ОС, браузерах или с определёнными версиями ПО. |
| Дефект локализации         | Ошибки перевода, некорректное отображение текста на разных языках, неправильные форматы дат, валют. |
| Дефект интеграции          | Ошибки во взаимодействии с внешними системами, сервисами, API. |
| Дефект установки           | Проблемы при установке, обновлении или удалении приложения. |
| Дефект логики              | Ошибки в бизнес-логике, неправильные расчёты, некорректные условия переходов. |
| Дефект данных              | Потеря, искажение, дублирование или некорректное отображение данных. |
| Дефект навигации           | Некорректная работа ссылок, кнопок, переходов между экранами. |
| Дефект доступности         | Недоступность приложения для людей с ограниченными возможностями (например, отсутствие alt-тегов, невозможность навигации с клавиатуры). |
| Дефект документации        | Ошибки, неточности или отсутствие документации для пользователя или разработчика. |
| Дефект конфигурации        | Ошибки в настройках приложения, невозможность изменить параметры, некорректная работа с разными конфигами. |
| Дефект синхронизации       | Проблемы с обновлением данных между клиентом и сервером, рассинхронизация данных. |
| Дефект времени             | Ошибки, связанные с обработкой времени, дат, временных зон, таймеров. |
| Дефект безопасности данных | Несоблюдение политики конфиденциальности, утечка персональных данных. |
| Дефект восстановления      | Проблемы с восстановлением после сбоев, отсутствие резервного копирования. |
| Дефект масштабируемости    | Проблемы при увеличении нагрузки, невозможность обработки большого количества пользователей или данных. |
| Дефект регресса            | Возникновение старых, ранее исправленных ошибок после внесения изменений. |
| Дефект совместного доступа | Ошибки при одновременной работе нескольких пользователей с одними и теми же данными. |
| Дефект уведомлений         | Некорректная работа push-уведомлений, email-уведомлений, SMS и других каналов оповещения. |
| Дефект лицензирования      | Проблемы с активацией, проверкой или истечением лицензии. |
| Дефект тестовых данных     | Использование некорректных или устаревших тестовых данных, влияющих на результаты тестирования. |
